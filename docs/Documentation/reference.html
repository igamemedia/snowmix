Reference Manuals

Updated for Snowmix 0.5.1

Introduction


This page list all the reference manuals for Snowmix. However some manuals are still work in progress.

The reference manuals are organized in groups as listed on the Reserved Commands page. In the reference manual each Snowmix command is listed with its name, its syntax and its parameters. Each command's effect is also explained and many commands are listed with example of usage.

The reference manuals will over time be written and ported to this web site, but for now, most reference manuals are found in the old Wiki system for Snowmix.

Overview


The refreserved commands can be split up into the following groups of commands:

  • Reference manual for Audio Feed commands.
  • Reference manual for Audio Mixer commands.
  • Reference manual for Audio Sink commands.
  • Reference manual for Feed commands.
  • Reference manual for General commands.
  • Reference manual for Image commands.
  • Reference manual for Macros and conditionals.
  • Reference manual for OpenGL Shape commands.
  • Reference manual for OpenGL Placed Shape commands.
  • Reference manual for Placed Shape commands.
  • Reference manual for Shape Commands.
  • Reference manual for Tcl Interpreter commands.
  • Reference manual for Text commands.
  • Reference manual for Virtual Feed commands.

Because reference manuals are a little dry to read and often do not reveal the greater picture, Snowmix Guides has been written for some of command groups.
